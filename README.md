# LingoAce 师资运营 · 诊断与优化方案

Selena 二面材料 — 师资运营诊断 Dashboard（Next.js）。

## 本地运行

```bash
npm install
npm run dev
```

- 主页：http://localhost:3000/teacher-ops
- 诊断计算器：http://localhost:3000/teacher-ops/calculator

## 部署

使用 [Vercel](https://vercel.com) 部署 Next.js 项目。`/teacher-ops` 为静态交互页，无需数据库。

## 技术栈

Next.js 15 · Tailwind CSS
